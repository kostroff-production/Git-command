### Begining
# Установка гит
```
apt install git
```
# Настройка гит
```
git config --global user.name "Your name here"
git config --global user.email "your_email@example.com"
git config --global color.ui true
```
#SSH настройка
Генерируем ключ <br>
`ssh-keygen -t rsa -C "your_email@example.com"` <br>
